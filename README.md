# Yana Engine

Educational cross-platform game engine written in C++23.

Goals:

- Learn Vulkan
- Learn modern C++
- Learn graphics programming
- Learn engine architecture

Primary platform:
Linux

Target platforms:
Linux
Windows
macOS

## Requirements

- CMake
- Ninja
- Clang (recommended)
- GCC (supported)

## Building

cmake --preset debug-clang

cmake --build --preset debug-clang
